Welcome

The "a" tag is called the link tag because the content of it has a link to someother page where it will redirect the user.

This is a link to My First Document Webpage. This is an another link to the "W3 Schools" Webpage.
An attribute called "href" is required to be used inside the "a" start tag to mention the location/url of the link we need.

The "img' tag is used to display any pictures from own system or available on the web. It has no end tag.
Attributes "src" defines the source of the picture.
"alt" is for any alternate text in case if the image fails to open.
"width" & "height" attributes are for defining the size of the picture.

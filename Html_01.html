The "p" Tag:

The "p" tag is called the paragraph tag and this is used for a sequence of content in a paragraph pattern or when you use the "p" tag it always starts from the new line.

The size H7 is the smallest of all the heading sizes while size H1 is the largest among them.

When there is some big paragraph.

Or there are more number of sentences making the paragraph a large one.

Or even when there is a need for a space between them.


The "br" Tag:

The "br" tag is for the line break and it doesn't have any end tag for it. And also the "b" tag is used to make the content bold.


The bold Tag:

This is a heading tag but the "b" tag doesn't work with it. But the "heading" itself is kind of bold letters only by default.
